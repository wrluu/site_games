# Интернет-магазин мебели

## Описание проекта

Интернет-магазин мебели — это веб-приложение, которое позволяет пользователям просматривать каталог мебели, добавлять товары в корзину и оформлять заказы. 

На сайте представлены различные категории мебели, такие как кровати, диваны, кресла, столы, шкафы и многое другое. Пользователи могут фильтровать товары по цене, цвету, типу комнаты и другим параметрам.

## Запуск проекта

1. Скачай или клонируй репозиторий с проектом.
2. Открой файл `index.html` в любом современном браузере (Chrome, Edge, Firefox и т.д.).
   
Если ты хочешь внести изменения в проект, открой файлы в текстовом редакторе или IDE (например, Visual Studio Code) и редактируй HTML или CSS.

## Используемые технологии

- **HTML** — структура сайта
- **CSS** — стилизация и адаптивная вёрстка
- **Google Fonts** — шрифт Montserrat
- **Медиазапросы** — адаптация под мобильные устройства и планшеты
